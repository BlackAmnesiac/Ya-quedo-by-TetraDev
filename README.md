# Ya Quedó - Marketplace de Servicios Locales

Plataforma web moderna que conecta clientes con trabajadores independientes de confianza en Latinoamérica.

## Características

- **Diseño Responsive**: Adaptado a todos los dispositivos
- **Iconos Profesionales**: Font Awesome en lugar de emojis
- **Animaciones Interactivas**: Efectos hover y transiciones suaves
- **UI Moderna**: Gradientes, sombras y diseño profesional
- **Accesibilidad**: Navegación por teclado y semántica HTML5

## Tecnologías

- **HTML5**: Semántico y accesible
- **CSS3**: Variables CSS, Grid, Flexbox, Animaciones
- **JavaScript**: Interactividad y accesibilidad
- **Font Awesome**: Iconos profesionales

## Estructura del Proyecto

```
ya-quedo/
|-- index.html      # Página principal
|-- styles.css      # Estilos y diseño
|-- script.js       # Interactividad
|-- README.md       # Documentación
```

## Funcionalidades

- Navegación fija con menú hamburguesa para móviles
- Scroll suave entre secciones
- Animaciones al hacer scroll
- Efectos hover en todos los elementos interactivos
- Diseño completamente responsive
